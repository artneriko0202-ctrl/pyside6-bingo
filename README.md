# pyside6-bingo

PySide6 で作成したビンゴアプリケーションです。

## 機能

- 3×3 / 5×5 のビンゴボードに対応
- プレイヤー管理（作成・編集・削除）
- ビンゴテンプレートの作成・編集・管理
- セッション管理と結果履歴の記録
- ダーク／ライトテーマ切り替え
- SQLite によるデータ永続化
- 配信に常時映す為の機能は未実装(晒し行為で規約でまずいかも？)

## 必要環境

- Python 3.10+
- PySide6

## インストール

```bash
pip install PySide6
```

## 使い方

```bash
python main.py
```

## ファイル構成

| ファイル | 説明 |
|---|---|
| `main.py` | メインアプリケーション |
| `config.json` | テーマ・表示設定 |
