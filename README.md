# 🎯 分かってない勢ビンゴ

PySide6 で作ったビンゴアプリ。プレイヤーごとにビンゴカードを管理して、成績やランキングも自動で出る。

## ✨ 機能一覧

### ビンゴカード
- **3×3 / 5×5** のビンゴボードに対応
- セルクリックでマーク切替
- ビンゴ達成時のフラッシュアニメーション（金色に点滅）
- シャッフル機能（セルをランダムに並べ替え）
- マーク状態の自動保存・復元

### プレイヤー管理
- プレイヤーの作成・名前変更・削除
- プレイヤーごとのメモ機能

### テンプレート管理
- ビンゴテンプレートの作成・編集・削除
- テンプレートは全プレイヤー共通で使える
- グリッド形式でセル内容を編集

### 📊 プレイヤー統計
- セッション数 / ビンゴライン数 / マーク率 / レーティング / 順位を表示
- 成績に応じた分析コメントを自動生成（カジュアルな日本語）
- マーク済みテキストの一覧（全体集計 + セッション別タブ）
- プレイヤーへのメモ欄

### 🏆 ランキング・レーティング
- レーティング自動算出（ビンゴ効率 × 500 + マーク率 × 200 + ライン数 × 20）
- **8段階のランク帯**: S+ / S / A+ / A / B+ / B / C / D
- 総合ランキング + テンプレート別ランキング

### セッション管理
- プレイヤー × テンプレートの組み合わせでセッションを自動復元
- 達成取消・セッション削除

### 🎨 テーマ・設定
- ダーク / ライトテーマ切り替え
- フォントサイズ変更
- ウィンドウサイズの自動保存

## 必要環境

- Python 3.10+
- PySide6

## インストール

```bash
pip install PySide6
```

## 使い方

```bash
python main.py
```

起動すると **プレイヤー選択 → テンプレート選択（or 新規作成） → ビンゴ開始** の流れで始まる。

## exe ビルド

```bash
pip install cx_Freeze
python setup.py build
```

`build/` フォルダに `分かってない勢ビンゴ.exe` が生成される。

## ファイル構成

| ファイル | 説明 |
|---|---|
| `main.py` | メインアプリケーション |
| `config.json` | テーマ・表示設定 |
| `setup.py` | cx_Freeze ビルド設定 |
| `requirements.txt` | 依存パッケージ |

## データベース

SQLite（`bingo.db`）で以下を管理：

| テーブル | 内容 |
|---|---|
| `players` | プレイヤー情報・メモ |
| `bingo_templates` | ビンゴテンプレート定義 |
| `bingo_cells` | テンプレートの各セルテキスト |
| `bingo_sessions` | プレイ記録（状態・日時） |
| `bingo_results` | セッション結果（マーク・ライン・ボード配置） |
